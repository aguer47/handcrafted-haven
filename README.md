# Handcrafted Haven

Handcrafted Haven is a web application that provides a platform for artisans and 
crafters to showcase and sell their unique handcrafted items. It connects talented 
creators with customers who appreciate handmade, sustainable products.

## Features

✅ **Completed Features:**
- Navigation bar with browse, sell, about, login, and sign up links
- Hero banner with call to action
- Featured products section with images and seller info
- Shop by category section
- Featured artisans section
- Customer reviews section
- Footer with quick links
- User registration and login pages
- Seller profile creation page
- Add product page
- Browse products page with search and filtering
- Product detail page with ratings and reviews
- Fully responsive design

📋 **Planned Features:**
- Edit seller profile
- Edit and delete products
- Backend integration with database
- User authentication
- Shopping cart
- Checkout process

## Getting Started

First, install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
